Game Info

Title: Chaddius Maximus
Released: June 14, 2019
Platforms: Windows
Genre Tags: Singleplayer, Adventure, 2D, Platformer, Metroidvania, Pixel Art

Role: Level Designer
Duration: 8 Weeks
Engine: Coyote (Custom Engine)
Tools Used: Tiled Map Editor

Download Link: https://igad.itch.io/chaddiusmaximus

Chaddius Maximus is a 2D Platformer Metroidvania where you take control of the big-muscled hero Chaddius, who must save his kingdom from a spreading corruption. With responsive movement and a throwable spear, players must traverse the ruins of the castle and fight the monsters that reside within.

Chaddius Maximus was created using the Coyote Engine, a custom made engine by a team of programmers in 16 weeks. In those last 8 weeks, the team was reformed and development for Chaddius Maximus started. Before that, while the engine was being developed, I was working on the Skaturion project (Coming Soon).

All levels were made using the free level editor called Tiled. With intergration set up by the programming team, level designers were able to easily create and efficiently iterate on levels built using square tiles. By cleverly utilizing parallax, layers, and transparency, we were able to create interesting gameplay and pleasing visuals.

My biggest contributions to this project were the forest levels at the beginning and in the middle of the game. The first few levels were meant for onboarding and introducing the different enemy types, whereas the other levels, nearing the end of the game, introduced the Spear Throw ability and its functions.
